درباره من

من امیر علی ساکی هستم، دانش‌آموز پایه یازدهم رشته کامپیوتر از شهر دورود. از همون روزهای اولی که با دنیای وب آشنا شدم، فهمیدم که طراحی و توسعه سایت فقط یه مهارت نیست—یه مسیر برای خلق، یادگیری و رشد شخصیه. با تمرکز روی HTML، CSS، JavaScript و C# تونستم پروژه‌های واقعی بسازم و اون‌ها رو روی GitHub منتشر کنم. هدف من فقط یادگیری نیست؛ من دنبال ساختن چیزهایی هستم که دیده بشن، تأثیر بذارن و تجربه کاربری عالی داشته باشن. عاشق چالش‌های فنی‌ام و همیشه دنبال اینم که سطح کارم رو بالاتر ببرم. این سایت رزومه، فقط یه صفحه نیست—یه قدم جدی برای ورود حرفه‌ای به دنیای فناوریه.

خدمات من

توسعه وب مدرن

طراحی و پیاده‌سازی صفحات وب واکنش‌گرا با استفاده از HTML، CSS، JavaScript و فریم‌ورک‌های روز. تمرکز بر تجربه کاربری، سرعت بارگذاری و امنیت.

برنامه‌نویسی با C#

ساخت نرم‌افزارهای دسکتاپ با Windows Forms و WPF، طراحی فرم‌های حرفه‌ای، مدیریت داده‌ها و اتصال به پایگاه‌داده با ADO.NET و Entity Framework.

مدیریت و تحلیل داده‌ها

طراحی فرم‌های اطلاعاتی، مدیریت پایگاه داده با SQL Server و ساخت گزارش‌های تحلیلی با ابزارهایی مثل Excel و Access برای تصمیم‌گیری بهتر.
